#  Mini E-Commerce System 

## What is this project?

This is a simple shopping system made using Java.
It works in the command line (terminal) and helps simulate how basic online shopping works.

You can:

* Add products
* View products
* Add items to a cart
* Generate a bill

Think of it like a **version of an online shopping app**, but running in your computer terminal.


##  Why was this project made?

This project was created to:

* Understand how real shopping systems work
* Practice Java programming in a practical way
* Learn how to manage data like products and cart items

It is mainly for learning purposes and demonstrates core programming concepts.



##  Who can use this?

Anyone can run this project:

* Students learning Java
* Even non-technical users (just follow steps below)


##  What does it do?

The system allows you to:

1. Add your own products (name, price, ID)
2. View all available products
3. Select products and add them to cart
4. Enter quantity
5. Get a final bill with GST


##  How does it look?

When you run the program, you will see a menu like this:

1. Add Product
2. View Products
3. Add to Cart
4. Checkout
5. Exit

You just enter a number to choose what you want to do.


##  Example Usage

* Add a product (Laptop, ₹50000)
* Add it to cart with quantity 2
* System will show:

Laptop x2 = ₹100000
GST (18%) = ₹18000
Total = ₹118000


##  How is it built?

The project is divided into simple parts:

* Product → stores product details
* Cart → manages shopping items
* CartItem → stores product + quantity
* Main → handles user interaction


##  Concepts Used

* Java Basics
* Object-Oriented Programming (OOP)
* ArrayList (for storing data)
* User Input (Scanner)


##  Important Notes

* Make sure you add products before adding to cart
* Enter correct product ID while adding to cart
* Quantity should be greater than 0
